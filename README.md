# React-image-gallery

To run the app

```
git clone <repo>
cd react-image-gallery
```

- Add your `REACT_APP_FLICKR_API_KEY` key in .env file

## To install all packages
```
npm i | yarn
```
## To run the project
```
npm start | yarn start
```
